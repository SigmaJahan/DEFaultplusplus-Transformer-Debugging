# Stage 3: Root-Cause Diagnosis

This package implements **Stage 3** of the FrankenFormer pipeline: root-cause diagnosis via differential signature analysis and Neural Diagnosis Graph (NDG) construction.

## Directory Structure

```
3_A_Diagnosis_Root_Cause/
  ndg_stage3/                        # NDG graph builder module
    cli.py                           # CLI entry point (python -m ndg_stage3.cli)
    ndg.py                           # Core NDG graph building
    mapping.py                       # Feature-to-subsystem mapping
    inference.py                     # NDG scoring reference implementation
    plotting.py                      # Publication-grade PDF plots
    feature_core_map.md              # Authoritative feature-subsystem mapping (146 features, 14 subsystems)
    generate_thesis_figures.py       # Thesis figure generation from Stage 1/2 JSONs
    generate_stage12_paper_plots.py  # Paper-ready Stage 1/2 plots
  ndg_diagnosis_builder/             # Mutation-grounded diagnosis builder
    run_stage3_diagnosis_ndg_v1.py   # Generates enc/dec_diagnosis.json
    DIAGNOSIS_BUILDER_README.md      # Usage documentation
  run_ndg_stage3.sh                  # Full pipeline runner
  run_ndg.sh                         # Simple CLI wrapper
  requirements.txt                   # Dependencies (matplotlib, numpy)
  stage_3.md                         # Mathematical specification of NDG scoring
```

## Inputs

Stage 1/2 results (from `../results/`):
- `stage_1_detection/enc_detection.json`, `dec_detection.json`
- `stage_2_categorization/enc_categorization.json`, `dec_categorization.json`
- `stage_2.1_categorization_xai/xai_enc_categorization.json`, `xai_dec_categorization.json`

Feature mapping:
- `ndg_stage3/feature_core_map.md`

Optional (mutation-grounded priors):
- `enc_diagnosis.json`, `dec_diagnosis.json` (generated by `ndg_diagnosis_builder/`)

## Outputs

Written to `../results/stage_3_diagnosis/`:
- `ndg_schema.json`, `ndg_encoder.json`, `ndg_decoder.json`
- `ndg_provenance.md`
- `enc_diagnosis.json`, `dec_diagnosis.json`
- `plots/` (PDF figures, if `--plots` enabled)

## Run

```bash
# Full pipeline (builds NDG from existing Stage 1/2 results)
bash run_ndg_stage3.sh

# Or manually via CLI
pip install -r requirements.txt
python -m ndg_stage3.cli \
  --enc_detection     ../results/stage_1_detection/enc_detection.json \
  --enc_categorization ../results/stage_2_categorization/enc_categorization.json \
  --xai_enc           ../results/stage_2.1_categorization_xai/xai_enc_categorization.json \
  --dec_detection     ../results/stage_1_detection/dec_detection.json \
  --dec_categorization ../results/stage_2_categorization/dec_categorization.json \
  --xai_dec           ../results/stage_2.1_categorization_xai/xai_dec_categorization.json \
  --feature_core_map  ndg_stage3/feature_core_map.md \
  --out_dir           ../results/stage_3_diagnosis \
  --plots
```

## NDG Scoring

`inference.py` implements the reference scoring function:

Score(f|x) = log prior(f) + alpha * sum_core w_shap(core) * phi(z_core) + beta * sum_sub impact_f(sub) * phi(z_sub)

See `stage_3.md` for the full mathematical specification.
